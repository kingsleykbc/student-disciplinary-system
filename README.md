# student-disciplinary-system
A full php student management and disciplinary system. This project has basic php Create, Remove, Update and Delete (CRUD) functionalities. The data is stored in a MYSQL database, and communication is done through PHP and AJAX requests.
Features implemented include
1. Live search
2. Adding, deleting and updating data through ajax calls
3. Log tracking
4. Login and registration

# instructions
1. Clone or download the repo
2. move the files to C:\wamp\www
2. start wamp server and open 'localhost/phpmyadmin'
3. import the 'sds.sql' file
4. open 'localhost/project/login.php'
